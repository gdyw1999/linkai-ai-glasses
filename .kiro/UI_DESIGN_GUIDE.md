# Linkai星韵AI眼镜 UI设计指南

## 设计理念
- **现代化**: 采用Material Design 3规范
- **专业感**: 深色顶部栏 + 浅色背景
- **易用性**: 清晰的视觉层级和交互反馈

## 色彩系统

### 主色调
- **深灰色**: `#1F1F1F` - 顶部栏背景
- **浅灰色**: `#F8F9FA` - 页面背景
- **白色**: `#FFFFFF` - 卡片背景

### 功能色
- **蓝色**: `#2196F3` - 拍照按钮
- **红橙色**: `#FF5722` - 录像按钮
- **紫色**: `#9C27B0` - 录音按钮
- **绿色**: `#4CAF50` - 智能识图按钮
- **橙色**: `#FF9800` - 扫描按钮

### 状态色
- **成功绿**: `#2E7D32` - 已连接
- **警告橙**: `#E65100` - 未连接
- **错误红**: `#C62828` - 录制中

## 组件规范

### 1. 顶部状态栏
```
高度: 80dp
背景: #1F1F1F (深灰色)
圆角: 下方16dp
内容:
  - 左侧: 应用名称 + 副标题
  - 右侧: 连接状态指示器
```

### 2. 卡片 (Card)
```
圆角: 12dp
阴影: 2dp (默认), 4dp (按钮)
内边距: 16-20dp
背景: 根据内容类型选择
```

### 3. 按钮
```
高度: 44dp (普通), 110dp (快捷功能)
圆角: 8dp (普通), 12dp (快捷功能)
阴影: 4dp (默认), 8dp (按下)
字体: Bold, 14sp
```

### 4. 快捷功能按钮
```
尺寸: 110dp 高度
排列: 2x2 网格
间距: 12dp
图标: 36sp
标签: 12sp Bold
颜色: 各功能独立色彩
```

## 间距规范

- **页面边距**: 16dp
- **卡片间距**: 24-32dp
- **组件间距**: 12dp
- **内部间距**: 16-20dp

## 字体规范

- **标题**: 18-20sp, Bold
- **副标题**: 14-16sp, SemiBold
- **正文**: 13-14sp, Regular
- **标签**: 11-12sp, Regular/Bold

## 页面布局模板

```
┌─────────────────────────────────┐
│   顶部状态栏 (深灰色)            │
│   - 应用名称 + 连接状态          │
└─────────────────────────────────┘
│ 16dp 边距                        │
├─────────────────────────────────┤
│  主要内容卡片                    │
│  - 连接状态卡片                  │
│  - 快捷功能网格                  │
│  - 其他功能区域                  │
├─────────────────────────────────┤
│ 16dp 边距                        │
└─────────────────────────────────┘
```

## 实现检查清单

每个新页面实现时，需要检查：

- [ ] 使用深灰色顶部栏 (`#1F1F1F`)
- [ ] 页面背景为浅灰色 (`#F8F9FA`)
- [ ] 卡片圆角为 12dp
- [ ] 按钮使用对应的功能色
- [ ] 间距遵循 16dp/12dp 规范
- [ ] 字体大小和粗细符合规范
- [ ] 状态指示器使用对应的状态色
- [ ] 所有交互元素有阴影和按下效果
