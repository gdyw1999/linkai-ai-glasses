# 设备详情对话框实现说明

## 功能概述

根据官方App截图（home03.jpg），实现了点击首页设备信息后弹出的设备详情对话框，显示设备的完整信息。

## 实现内容

### 1. 设备详情对话框 (DeviceDetailsDialog)

**触发方式**:
- 点击首页连接状态卡片中的设备信息区域（设备名称和电量部分）

**显示内容**:
- ✅ 设备名称
- ✅ MAC地址（可点击复制到剪贴板）
- ✅ 电量信息（百分比 + 充电状态）
- ✅ 设备型号（Linkai星韵AI眼镜）
- ✅ SDK版本（1.1.0）
- ✅ 固件版本（待从SDK获取）
- ✅ 连接状态图标

### 2. UI设计特点

**对话框样式**:
- 使用 `AlertDialog` 实现
- 圆角 16dp
- 标题栏：设备详情 + 绿色连接图标
- 内容区：可滚动的设备信息列表
- 底部按钮：蓝色"关闭"按钮

**信息项布局**:
- 图标（24dp）+ 标签 + 值
- 使用分隔线区分不同信息
- 图标颜色：蓝色 (#2196F3)
- 标签字体：12sp，灰色
- 值字体：14sp，加粗，深灰色

**电量显示**:
- 根据状态显示不同颜色的图标：
  - 充电中：橙色 (#FFA726)
  - 正常电量：绿色 (#4CAF50)
  - 低电量：红色 (#C62828)
- 显示百分比 + 充电状态文字

**交互功能**:
- 点击MAC地址复制到剪贴板
- 显示Toast提示"已复制: [MAC地址]"
- 点击"关闭"按钮或对话框外部关闭

### 3. 代码结构

```kotlin
// HomeScreen.kt

// 1. 添加对话框显示状态
var showDeviceDetailsDialog by remember { mutableStateOf(false) }

// 2. 在ConnectionStatusCard添加点击事件
ConnectionStatusCard(
    ...
    onDeviceClick = { showDeviceDetailsDialog = true }
)

// 3. 显示设备详情对话框
if (showDeviceDetailsDialog && uiState.isConnected) {
    DeviceDetailsDialog(
        deviceName = uiState.deviceName,
        batteryLevel = uiState.batteryLevel,
        isCharging = isCharging,
        onDismiss = { showDeviceDetailsDialog = false }
    )
}

// 4. 设备详情对话框组件
@Composable
fun DeviceDetailsDialog(...)

// 5. 设备详情项组件
@Composable
fun DeviceDetailItem(...)
```

### 4. 数据来源

| 信息项 | 数据来源 | 状态 |
|--------|---------|------|
| 设备名称 | HomeViewModel.uiState.deviceName | ✅ 已实现 |
| MAC地址 | GlassesSDKManager.getCurrentDevice().address | ✅ 已实现 |
| 电量百分比 | HomeViewModel.uiState.batteryLevel | ✅ 已实现 |
| 充电状态 | HomeViewModel.isCharging | ✅ 已实现 |
| 设备型号 | 硬编码 | ✅ 已实现 |
| SDK版本 | 硬编码（1.1.0） | ✅ 已实现 |
| 固件版本 | 待从SDK获取 | ⏳ 待实现 |

## 使用方法

### 用户操作流程

1. 打开App，连接眼镜设备
2. 在首页看到连接状态卡片（绿色背景）
3. 点击设备信息区域（设备名称和电量部分）
4. 弹出设备详情对话框
5. 查看设备详细信息
6. 点击MAC地址可复制到剪贴板
7. 点击"关闭"按钮或对话框外部关闭

### 开发者集成

```kotlin
// 在HomeScreen中已经自动集成
// 只需确保设备已连接即可使用

// 如果需要在其他页面使用，可以复用DeviceDetailsDialog组件
DeviceDetailsDialog(
    deviceName = "设备名称",
    batteryLevel = 85,
    isCharging = false,
    onDismiss = { /* 关闭逻辑 */ }
)
```
