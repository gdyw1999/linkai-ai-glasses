**项目**: 青橙AI眼镜App

---

## MVP定义

MVP (Minimum Viable Product) 最小可行产品，聚焦核心用户场景，用最少功能验证产品价值。

### MVP目标
- 验证眼镜设备与App的连接稳定性
- 验证核心功能(拍照/录像/录音/AI对话/同步)的可用性
- 验证LinkAI语音交互体验
- 收集用户反馈，为后续迭代提供依据

## 核心功能设计

### 1. 设备连接模块

| 功能 | API | 说明 |
|------|-----|------|
| 扫描设备 | `BleScannerHelper.scanDevice()` | 扫描眼镜 |
| 连接设备 | `BleOperateManager.connectDirectly()` | 连接指定设备 |
| 断开连接 | `BleOperateManager.unBindDevice()` | 断开连接 |
| 连接状态 | `GlassesDeviceNotifyListener` | 监听连接/断开 |
| 电量同步 | `LargeDataHandler.syncBattery()` | 获取电量 |
| 版本信息 | `LargeDataHandler.syncDeviceInfo()` | 获取版本 |

### 2. AI对话模块

| 步骤 | LinkAI API | 说明 |
|------|------------|------|
| 1. 语音识别 | `POST /v1/audio/transcriptions` | 音频→文字 |
| 2. 对话生成 | `POST /v1/chat/memory/completions` | 文字→文字(带记忆) |
| 3. 语音合成 | `POST /v1/audio/speech` | 文字→音频 |

**对话流程**:
```
用户说话 → 眼镜录音 → App获取音频
    → LinkAI ASR (语音转文字)
    → LinkAI LLM (对话生成)
    → LinkAI TTS (文字转语音)
    → 眼镜播放语音 + App显示文字
```

### 3. 媒体同步模块

| 功能 | API | 说明 |
|------|-----|------|
| 初始化 | `GlassesControl.initGlasses(path)` | 初始化下载路径 |
| 开始同步 | `GlassesControl.importAlbum()` | 触发WiFi同步 |
| 同步进度 | `WifiFilesDownloadListener.fileProgress()` | 单文件进度 |
| 下载完成 | `WifiFilesDownloadListener.fileWasDownloadSuccessfully()` | 单文件完成 |
| 全部完成 | `WifiFilesDownloadListener.fileDownloadComplete()` | 全部下载完成 |

### 4. 会话管理模块

**存储**: SQLite / Room

**表结构**:

Table: conversations
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 主键 |
| title | String | 会话标题 |
| created_at | Long | 创建时间 |
| updated_at | Long | 更新时间 |

Table: messages
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 主键 |
| conversation_id | String | 外键 |
| role | String | user/assistant |
| content | String | 文字内容 |
| audio_path | String | 本地音频路径（可选） |
| created_at | Long | 时间 |

## 技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 框架 | Android (Kotlin) | minSdk 24, targetSdk 35 |
| UI | Jetpack Compose | 现代化声明式UI |
| 网络 | OkHttp + Retrofit | HTTP请求 |
| 数据库 | Room | 本地会话存储 |
| 蓝牙 | 青橙SDK (LIB_GLASSES_SDK-release.aar) | 设备控制 |
| AI | LinkAI API | ASR + LLM + TTS |
| 图片加载 | Coil | 媒体缩略图 |
| 权限 | XXPermissions | 权限请求 |
